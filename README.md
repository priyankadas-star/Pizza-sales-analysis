*Pizza Sales Analysis

## Project Overview
Analyzed pizza sales data using Excel, Python, and Power BI to identify revenue drivers, customer demand patterns, and top-performing products.

## Tools Used
- Excel
- Python (Pandas, Matplotlib)
- Power BI

## Key Insights
- Calculated Total Revenue
- Identified Top Selling Pizzas
- Analyzed Revenue by Category
- Created Interactive Dashboard

## Project Structure
- Dataset
- Python Analysis
- Power BI Dashboard
- Visualizations
